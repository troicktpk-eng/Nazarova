# Random Task Generator

Автор: Назарова Мария Александровна

## Описание
Приложение генерирует случайную задачу из предопределенного списка, позволяет фильтровать задачи по типу и сохраняет историю генераций. Вы можете добавлять новые задачи вручную.

## Инструкция по использованию
- Нажмите «Сгенерировать задачу» для получения случайной задачи.
- Используйте фильтр по типу для отображения задач определенного типа.
- Введите новую задачу и выберите ее тип, чтобы добавить ее в список задач.
- История задач сохраняется автоматически и загружается при запуске.

## Примеры тестов
- Генерация случайной задачи.
- Фильтрация задач по типу.
- Добавление новой задачи и проверка отображения.
